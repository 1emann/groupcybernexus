🔍 Understanding DNS, IPs & Protocols

📌 Domain Name System (DNS)

DNS translates **human-friendly domain names** (e.g., example.com) into **IP addresses**.

Example:

example.com ➝ 192.168.1.1

🖥️ IP Addresses

Each device connected to the internet has an IP address, which can be **IPv4 (e.g., 192.168.1.1)** or **IPv6 (e.g., 2001:db8::ff00:42:8329)**.

🔗 Networking Protocols

  • 📩 HTTP/HTTPS - Web communication protocols (HTTPS is encrypted for security).
  • 📡 TCP/IP - Foundation of internet communication.
  • 🔑 SSL/TLS - Ensures secure web browsing.

🛠️ Test Networking in Action

Use these commands in the **terminal** to analyze website networking:

🖥️ Find a Website’s IP Address

nslookup example.com

📶 Check If a Website is Online

ping example.com

📜 View HTTP Headers

curl -I https://example.com

🔐 Web Security & Best Practices

  • ✅ Always use **HTTPS** to encrypt web traffic.
  • ✅ Use **firewalls** to protect against cyber threats.
  • ✅ **DNSSEC** helps prevent domain hijacking.
